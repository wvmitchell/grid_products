## Grid products

### Description

This program will compute the products of numbers held within a MxN grid. It's
possible to recieve calculations in a horizontal, vertical, or diagonal
direction.

  12 23 34
  **34** **44** **33** = 49368

  22 98 83

  23 **12** 07
  23 **85** 93 = 1020
  32 **01** 39

  23 **10** 23
  22 09 **84** = 840
  43 23 45
